Maintenance

The final phase of implementation, in which all subsequent new courses are developed for the new system, will transition into the maintenance phase. HR personnel will be trained (incidentally, with the new system) to give any required support to users of the system and to monitor it for any anomalies. The maintenance phase is arguably the most important of all, as maintainers will ensure continued compatibility and course relevance.

Chrome Updates

An important part of the maintenance process for this project will be the task of updating Google Chrome. Maintainers will need to:

  • Apply critical security updates
  • Test feature updates for compatibiity
  • Apply feature updates if they pass compativility tests
  • Request developer to create workaround or wait for compatible update if compatibility tests fail

Course Updates

The content of the training courses eventually lose their relevance and must be updated or a new course must be created. Maintainers will need to:

  • Determine if a course should be updated or recreated
  • Turn requests for courses from management into RFPs for content creators
  • Review responses and select content creator for a given course
