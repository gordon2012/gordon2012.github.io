Problems with CBL

There have been many complaints about the aging CBL system caused by Internet Explorer and Microsoft's existing continued support but lack of real functional updates for older versions. This has lead to an increase in the number of tickets being created and results in wasted IT resources that might overwise solve more important problems. Some of these problems include:

  • Internet Explorer crashing
  • The thin client crashing
  • Course progress not being saved
  • Courses not visible that should be

Stakeholders

The idea for the project originated from a large supercenter in Southern California whose HR manager was frustrated with the CBL system. The stakeholders for this project include:

  • Champion: HR Manager
  • Store Manager
  • Sponsor: Regional HR Director

Project Description

The project entails the installation of a modern web browser and the creation of new course content created in HTML5. The project will involve:

  • The replacement of Internet Explorer with Google Chrome
  • The replacement of Flash with HTML5 with the possibility of lightweight web frameworks

Project Scope

The scope for this project is limited to the CBL system, but the HR program director has indicated interest in expanding the scope to other projects using similar technologies under the HR umbrella if this project is successful.
