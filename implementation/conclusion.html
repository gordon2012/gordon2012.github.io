Conclusion

This project consists of the design and implementation of a new Computer Based Learning system for Walmart to use to train its employees. It uses Google Chrome to access learning materials designed in basic HTML5 with possible web frameworks and will replace Internet Explorer accessing Flash based training content. It will be implemented in stages with the old system still running in parallel and courses will be replaced as needed until all courses use the new system.
